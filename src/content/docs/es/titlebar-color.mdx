Existe una clase llamada `TitlebarColor` que permite configurar los colores de la barra de título.

## Colores predefinidos

La clase `TitlebarColor` tiene varios colores predefinidos que se pueden usar para configurar los colores de la barra de título.

## Crear un color personalizado

Para crear un color personalizado, use el método estático `TitlebarColor.fromHex(color)`. Este método acepta un color hexadecimal como argumento y devuelve un objeto `TitlebarColor`.

```js
const customColor = TitlebarColor.fromHex('#FF0000')
```
